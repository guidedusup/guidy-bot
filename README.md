# Guidy - Discord bot for GuideduSup

## Installation locale
1. Copier `.env.example` en `.env` et remplir `TOKEN` et `CLIENT_ID`.
2. `npm install`
3. `npm start`

## Déploiement
- Render: push sur GitHub et utiliser `render.yaml`. Ajouter `TOKEN` et `CLIENT_ID` dans les variables d'environnement du service.
- Replit: créer un projet Node, ajouter `TOKEN` en secret, puis `npm install` et `npm start`.

## Sécurité
- Ne partage jamais le `TOKEN`.
- Si le token fuit, régénère-le dans le Discord Developer Portal (Bot -> Regenerate).
